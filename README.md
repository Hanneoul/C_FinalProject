# \# 🕹️ C Command Battle AI Challenge Readme

# 

# \## 📝 1. 프로젝트 개요

# 

# 본 프로젝트는 C 언어와 printf/scanf 기반의 콘솔 환경에서 진행되는 2인용 \*\*동시 턴제 전략 게임\*\*입니다. 참가자(학생)는 외부 AI 파일을 작성하여 Player의 행동을 제어해야 하며, \*\*Quiz 시스템\*\*을 통해 고급 스킬을 해금해야 합니다.

# 

# \* \*\*목표\*\*: 상대방의 HP를 0으로 만듦.

# \* \*\*맵 크기\*\*: 15 (가로) × 7 (세로).

# \* \*\*자원\*\*: 초기 \*\*HP 5\*\*, 초기 \*\*MP 5\*\*.

# 

# ---

# 

# \## 🔑 2. 보안 및 AI 등록 규칙

# 

# \### A. Key 인증 및 API

# 

# 1\.  \*\*Key 획득\*\*: AI 등록 함수(\*\*`register\_player\_ai`\*\*) 호출 시, \*\*고유 `reg\_key`\*\*를 반환받아 저장해야 합니다.

# 2\.  \*\*해금/조회 API\*\*: 모든 스킬 해금(\*\*`attempt\_skill\_unlock`\*\*) 및 상태 조회(\*\*`is\_skill\_unlocked`\*\*)는 이 고유 `reg\_key`를 인증 수단으로 사용해야 합니다.

# 3\.  \*\*퀴즈 시스템\*\*: 고급 스킬은 \*\*`quiz\_data.csv`\*\*에 저장된 \*\*정답 문자열\*\*을 제출해야 해금됩니다.

# 

# \### B. 스킬 잠금 및 페널티

# 

# \* \*\*잠금 상태\*\*: 고급 스킬은 처음에 잠겨 있습니다.

# \* \*\*페널티\*\*: \*\*잠긴 스킬\*\*을 사용하려고 시도할 경우, 스킬은 실행되지 않고 \*\*사용자 자신에게 HP 2 피해\*\*가 즉시 적용됨.

# 

# ---

# 

# \## ⚔️ 3. 턴 진행 순서 및 규칙

# 

# 매 턴은 \*\*동시 입력\*\* 후 \*\*순차적 처리\*\*를 통해 공정하게 진행됨.

# 

# \### A. 턴 처리 순서

# 

# 1\.  \*\*DoT Phase\*\*: 독(Poison) 상태일 경우 HP 1 피해 적용.

# 2\.  \*\*Turn Priority 결정\*\*: 턴 시작 시 \*\*50% 확률\*\*로 P1/P2 중 한 명에게 턴 전체의 \*\*선공권\*\*이 부여됨.

# 3\.  \*\*Move Phase\*\*: 모든 이동(기본 이동, 점멸)이 처리됨. 충돌 시 선공권자만 이동 성공.

# 4\.  \*\*Magic Phase\*\*: 마법/전략 공격(독, 가로/세로 공격)이 \*\*선공권자 → 후공권자\*\* 순으로 실행됨.

# 5\.  \*\*Action Phase\*\*: 물리 공격(강타, 기본 공격, 자폭)이 \*\*선공권자 → 후공권자\*\* 순으로 실행됨.

# &nbsp;   > (후공권자는 선공권자의 공격을 맞고 \*\*생존해야만\*\* 반격 가능)

#
